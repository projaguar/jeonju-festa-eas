# 전주페스타 앱

## 목표

기존의 react-native (../JeonjuFesta )앱을 완전한 Expo App으로 포팅

## 주의사항

- Android, iOS 모두 기존 패키지명, 앱 이름, 인증서파일, 아이콘 등 모두 동일하게 적용하여 안전하게 배포 될 수 있어야 함.
- 필요없는 앱 권한 요청 설정은 제거.
- 모든 세부 기능 포팅
- expo 앱에 최적화 (사용하는 라이브러리는 모두 expo 용으로 교체)

## 변경사항

- 스플래시
  - 스플래시 화면은 변경해야함.
  - \_resources 의 splash 폴더에 사용할 리소스 있음
  - animation gif 또는 mp4 둘중에 한가지 더 적합한것을 사용하면 됨
  - 총 3종류 에니메이션이 있는데 실행될때 random 으로 3개중 한개 수행
